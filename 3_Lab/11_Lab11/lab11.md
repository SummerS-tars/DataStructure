## 1. 第一题：可以达到所有点的最小点集

### 1.1. 题目描述

- 一个 有向无环图，有 n 个节点编号为 0 到 n-1。给定一个边数组 edges，其中 edges[i] = [ai,bi] 表示一条从点 ai 到 bi 的有向边。
- 找到最小的出发点集：从这些点出发能到达图中所有点，删去任何一个点都无法到达图中所有点。
- 注：输入保证解存在且唯一。你可以以任意顺序返回这些节点编号。

### 1.2. 示例

Example1：  
输入: n = 6, edges = [[0,1],[0,2],[2,5],[3,4],[4,2]]  
输出: [0,3]  

Example2：  
输入: n = 5, edges = [[0,1],[2,1],[3,1],[1,4],[2,4]]  
输出: [0,2,3]  
解释：注意到节点 0，3 和 2 无法从其他节点到达，所以我们必须将它们包含在结果点集中，这些点都能到达节点 1 和 4 。  

## 2. 第二题：统计完全连通分量的数量

### 2.1. 题目描述

- 给你一个整数 n 。现有一个包含 n 个顶点的 无向图，顶点按从 0 到 n - 1 编号。给你一个二维整数数组 edges 其中 edges[i] = [ai, bi] 表示顶点 ai 和 bi 之间存在一条无向边。
- 返回图中 完全连通分量 的数量。
- 如果连通分量中每对节点之间都存在一条边，则称其为 完全连通分量 。
- 提示1: 首先使用合理的存储结构构建图。
- 提示2：对于有m个节点的完全连通分量，其边数为 m*(m-1)/2。

### 2.2. 示例

示例 1：  
输入： n = 6, edges = [[0,1],[0,2],[1,2],[3,4]]  
输出：3  

示例 2：  
输入：n = 6, edges = [[0,1],[0,2],[1,2],[3,4],[3,5]]  
输出：1  
解释：包含节点 0、1 和 2 的分量是完全连通分量，因为每对节点之间都存在一条边。 包含节点 3 、4 和 5 的分量不是完全连通分量，因为节点 4 和 5 之间不存在边。 因此，在图中完全连接分量的数量是 1 。  

## 3. 第三题：连接所有点的最小费用

### 3.1. 题目描述

- 给你一个 points 数组，表示 2D 平面上的一些点，其中points[i] = [ai,bi]。
- 连接点 [ai, bi] 和点 [aj, bj] 的费用为它们之间的曼哈顿距离 ：|ai - aj| + |bi - bj| ，其中 |val| 表示 val 的绝对值。
- 请你返回将所有点连接的最小总费用。只有任意两点之间有且仅有一条简单路径时，才认为所有点都已连接。

### 3.2. 示例

示例 1：  
输入：points = [[0,0],[2,2],[3,10],[5,2],[7,0]]  
输出：20  

示例 2：  
输入：points = [[3,12],[-2,5],[-4,1]]  
输出：18  

示例 3：  
输入：points = [[0,0],[1,1],[1,0],[-1,1]]  
输出：4
