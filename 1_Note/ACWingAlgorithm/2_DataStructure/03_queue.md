# 队列

Queue  
一种线性结构  
FIFO（First In First Out）  

## 数组模拟

在队尾插入元素，在队头弹出元素  

数组模拟表示法  

q\[N], hh, tt  

q\[N] 存储队列元素  
hh 队头指针  
tt 队尾指针  

操作：  

1. 初始化  
    hh = 0, tt = -1  

2. 插入  
    q\[++tt] = x;  

3. 弹出  
    hh ++;  

4. 判断队列是否为空  
    hh < tt  

5. 获取队头元素  
    q\[hh]  

6. 获取队尾元素  
    q\[tt]  
    *数组模拟专属*  

7. 获取队列中元素个数  
    tt - hh + 1  

8. 清空队列  
    hh = 0, tt = -1
