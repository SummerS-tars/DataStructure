# PJ2 优化建议

目前在游戏逻辑、丰富性、用户友好方面存在许多不足

1. 地图可视化  
    当前地图仅以文本形式展示，难以直观理解房间布局与连接关系。建议引入 ASCII 地图预览或图形化界面，展示房间位置、Boss 位置及怪物强度。  

2. 游戏逻辑  
    玩家与地图之间的关系目前尚不明确  
    建议引入一个类似实时互动的模式  

3. 游戏逻辑II  
    Boss移动应该是自发随机进行的而非目前由用户触发  
    此处逻辑应该改进  

4. 丰富性  
    目前战斗系统还非常原始和单一  
    考虑增加一点丰富性  

5. 用户友好性  
    当前 CLI 交互较为基础  

6. 测试逻辑问题  
    目前测试由于流程中包含菜单循环  
    导致无法自动完成完整流程（进入菜单后必须手动退出）  

7. 游戏逻辑III  
    装备逻辑（按道理对于某一类装备，只能装备一个？）
